Bringing the Library to the DC Public

Did you know you can search the DC Public Library catalog directly from your browser's search bar? Below are instructions for Chrome and Firefox users.

If you use Chrome

Visit the Chrome Settings menu and choose Manage search engines.

Click the Add button and a window will pop up.

Name your new search engine DC Public Library.

As keyword put dcpl (or something else if you'd prefer).

Finally, copy-paste the following into the URL field:

https://catalog.dclibrary.org/client/en_US/dcpl/search/results?qu=%s

When you're done the window should look like this:

To search the DCPL catalog you will:

  1. Type the Keyword you chose in the Chrome search bar
  2. Press the Space key
  3. Type your search terms (e.g. the book title or author)
  4. Press the Enter key

If you use Firefox

Under the Bookmarks menu, choose Show All Bookmarks.

Click the icon that looks like a gear and click New Bookmark

Name your new search engine DC Public Library.

Copy-paste the following into the location field:

https://catalog.dclibrary.org/client/en_US/dcpl/search/results?qu=%s

As keyword put dcpl (or something else if you'd prefer).

When you're done the window should look something like this:

To search the DCPL catalog you will:

  1. Type the Keyword you chose in the Firefox search bar
  2. Press the Space key
  3. Type your search terms (e.g. the book title or author)
  4. Press the Enter key
